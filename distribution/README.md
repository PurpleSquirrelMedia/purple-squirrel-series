## Platform-Specific Instructions

### Amazon KDP (Kindle)

1. Go to kdp.amazon.com
2. Create New Title → Kindle eBook
3. Upload EPUB from `epub/`
4. Upload cover from `covers/`
5. Set price (recommend $9.99-14.99)

**Metadata:**
- Publisher: Purple Squirrel Media
- Categories: Computers & Technology > Programming
- Keywords: programming, software development, AI, Claude, DevOps

### Apple Books

1. Use Apple Books for Authors or Transporter
2. Upload EPUB (cover embedded)
3. Set pricing and territories

### Draft2Digital

1. Go to draft2digital.com
2. Upload EPUB
3. Select distribution channels:
   - Apple Books
   - Barnes & Noble
   - Kobo
   - Libraries (OverDrive)
   - Scribd

### Gumroad (Direct Sales)

1. Create product at gumroad.com
2. Upload PDF
3. Set your own price
4. Keep 90%+ of revenue
