## The Purple Squirrel Series

A comprehensive collection for modern developers, covering everything from AI-assisted development to production infrastructure.

---

### Purple Squirrel Playbook
**Strategic Frameworks for Modern Development**

The definitive guide to building software in the AI era. Covers:

- **Part I: AI/LLM Development** — Claude integration, prompt engineering, agentic systems
- **Part II: Startup Strategy** — MVP development, architecture decisions, scaling
- **Part III: Crypto/Web3** — Blockchain security, wallet architecture, DeFi patterns

**Formats:** EPUB, PDF
**Pages:** ~80
**Best For:** Technical founders, senior developers, architects

---

### Purple Squirrel In a Nutshell
**A Desktop Quick Reference**

Keep this one open while you code. Essential commands and patterns at your fingertips:

- **Developer Tools** — Terminal, git, package managers, IDE shortcuts
- **API Reference** — REST, GraphQL, authentication, rate limiting
- **Language Cheatsheets** — Python, TypeScript, Rust, SQL

**Formats:** EPUB, PDF
**Pages:** ~60
**Best For:** All developers, quick lookup while coding

---

### Purple Squirrel Cookbook
**Solutions & Patterns for Every Problem**

Problem → Solution. Recipes you can copy and adapt:

- **Claude/AI Recipes** — Structured output, RAG, tool use, code review
- **Full-Stack Recipes** — Auth, WebSockets, file uploads, payments
- **DevOps Recipes** — Docker, CI/CD, monitoring, migrations

**Formats:** EPUB, PDF
**Pages:** ~70
**Best For:** Developers who learn by example
